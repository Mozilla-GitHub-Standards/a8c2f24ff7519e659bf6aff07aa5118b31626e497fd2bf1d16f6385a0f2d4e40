Build

Like desktop, mobile Firefox supplies a way for partners to repackage a set of customization files into a Firefox build without recompiling. The "distribution files" concept allows a small set of external files to be used to customize a standard Firefox build. Our wiki contains full details on creating your distribution files.

Distribute Firefox for Android

Mozilla supports three methods for distributing a custom bundled Firefox mobile browser:

Pre-Installation

 Pre-install Firefox to give users easy access to a mobile browser they’ll love, while also providing an integrated experience into your own services and offerings.

OTA Installation

 For reaching your existing user base, let us provide you with a unique URL to download your custom bundled browsing experience for an Over-The-Air (OTA) Firefox installation.

Alternative Store

 If you control another store as an alternative to Google Play, we are happy to discuss the opportunity to leverage this channel to reach your user base.

Custom Bundles

A custom partner bundle includes the general standard binaries in a digitally-signed APK plus additional distribution files that contains all of the agreed partner customization preferences. Upon the browser’s first run, the customizations get applied. When the standard APK is updated, only the core browser itself gets updated, thereby ensuring that the customizations are not over-written.
